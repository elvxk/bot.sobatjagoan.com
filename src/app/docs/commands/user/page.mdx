# User

### `/user info [opt]`
> ```
> Menampilkan informasi user sobat atau target user
> Jika tidak ada target user, maka akan menampilkan informasi user sobat
> ```
> [Option] `user`

### `/user avatar [opt]`
> ```
> Menampilkan avatar user sobat atau target user
> Jika tidak ada target user, maka akan menampilkan avatar user sobat
> ```
> [Option] `user`

### `/user banner [opt]`
> ```
> Menampilkan banner user sobat atau target user
> Jika tidak ada target user, maka akan menampilkan banner user sobat
> ```
> [Option] `user`
