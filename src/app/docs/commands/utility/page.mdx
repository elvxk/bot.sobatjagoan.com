# Utility

## Ping

### `/ping` 
> ```
> Cek latency ke server bot discord
> ```

## Help

### `/help`
> ```
> Melihat panel bantuan untuk bot discord 
> ```

## Status

### `/status [opt]`
> ```
> Melihat status dari beberapa service dari Balkon Project
> ```
>
> | Option | Description |
> |:----- | :------ |
> | `Minecraft` | Cek status server minecraft : Balkon MC
> | `FiveM` | Cek status server FiveM : Balkon FiveM
> | `SA-MP` | Cek status server GTA SAMP : Jagoan Roleplay

## Shift

### `/shift [opt]`
> ```
> Melihat shift atau jadwal dari team 24/7
> Jika tanpa option akan menampilkan jadwal semua divisi pada hari ini
> ```
>
> | Option | Description |
> |:----- | :------ |
> | `divisi` | Menampilkan jadwal pada divisi tertentu
> | `tanngal` | Menampilkan jadwal pada tanggal tertentu pada jadwal bulan saat ini
